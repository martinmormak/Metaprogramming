Metaprogramming Project

This project was developed as part of the Metaprogramming course.

It is a laboratory-based group assignment, where we implemented a software project focused on practical applications of metaprogramming concepts.

The work was based on topics covered in the course, including reflection, code generation, annotations, and other techniques for manipulating program structures as data.

The assignment required designing and implementing a solution that demonstrates the use of metaprogramming principles in practice, as well as working collaboratively as a team on the final project.
